# Mini Project 1 – Personal Budget Tracker (Excel)

Tracks personal expenses across seven categories (Food, Transport, Rent, Internet, Shopping, Electricity, Others) for Jan–Dec.

- Annual total: ₹3,35,602
- Highest spend month: October (₹32,075)
- Lowest spend month: August (₹24,967)

Files:
- `MINI_PROJECT_1.pdf` – Project report with explanation and charts
- `MINI_PROJECT_1.xlsx` – Excel workbook with data and formulas
